Architecture

GTFS to boardroom in one pipeline

Transit data is inherently real-time and high-volume. Our architecture handles both: streaming pipelines for operations teams, and batch forecasting models for planning teams — all feeding a unified Power BI environment.

Ingestion Layer

Azure Event Hubs ingesting GTFS-RT vehicle positions and service alerts. Azure Stream Analytics joining live feeds with scheduled timetable data in real time.

Analytics Layer

Azure ML for Prophet forecasting models. Python pipelines for Compass card revenue analysis. Delta Lake for efficient historical storage and time-series querying.
